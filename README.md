# CodeTunisiePro - Plateforme d'apprentissage du Code de la Route Tunisien

Bienvenue sur le frontend de **CodeTunisiePro**. Cette application est conçue pour aider les Tunisiens à apprendre et à s'entraîner pour l'examen du code de la route avec une interface moderne et intuitive.

## Identifiants de Test (Admin)

Si vous déployez cette version localement ou sur un serveur de test, vous pouvez utiliser les identifiants suivants pour accéder au panneau d'administration :

- **Email** : `admin@codetunisie.com`
- **Mot de passe** : `adminpassword123`

## Technologies Utilisées

- **React** avec **Vite**
- **Tailwind CSS** & **Radix UI** pour le design
- **i18next** pour le support multilingue (Français/Arabe)
- **Framer Motion** pour les animations

## Installation et Utilisation

1.  **Installation des dépendances** :
    ```bash
    npm install
    ```

2.  **Lancement du serveur de développement** :
    ```bash
    npm run dev
    ```

3.  **Build pour la production** :
    ```bash
    npm run build
    ```
